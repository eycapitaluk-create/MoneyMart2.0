# ラウンジ × ソーシャル投資 ベスト構成案

Trading212・eToro・X（Twitter）を参考にした、MoneyMartラウンジの改善方向性。

---

## 1. 各プラットフォームの強み

### Trading212
| 機能 | 内容 |
|------|------|
| **Pie** | Custom / Model / **Social Pie** の3種。Social = コミュニティ共有 |
| **共有フロー** | 作成 → 公開 → 他ユーザーがワンタップでコピー |
| **変更通知** | 配分変更時にコピーしたユーザーへ自動通知 |
| **UI** | シンプル、Pieが視覚的に中心 |

### eToro
| 機能 | 内容 |
|------|------|
| **CopyTrader** | 人気投資家を発見 → 選択 → COPY → 金額設定 |
| **発見** | 戦略タイプで検索（crypto / stock / ETF） |
| **透明性** | リターン・保有銘柄・リスクスコアを明示 |
| **コピーオプション** | 全ポジション or 新規トレードのみ |

### X（Twitter）
| 機能 | 内容 |
|------|------|
| **Smart Cashtags** | $TSLA, $BTC タップ → チャート・取引 |
| **コミュニティ** | Fintwit, Crypto X などハッシュタグで集約 |
| **トレンド** | 銘柄・トピックのトレンド表示 |
| **シームレス** | 発見 → 議論 → 取引まで一気通貫 |

## 2. ベスト構成の方向性

### A. ラウンジの役割を明確化
- **X寄り**: 発見・議論・トレンドのハブ
- **Trading212寄り**: ポートフォリオ共有・コピーの入口
- **eToro寄り**: 戦略別・リスク別の発見

→ **「発見 → 議論 → コピー」を一画面で完結**させる。

### B. サイドバー改善（Trading212 + eToro）

**現状**
- 相場観 / 話題の銘柄 / 人気ポートフォリオ

**改善案**
1. **人気ポートフォリオ**
   - 戦略フィルタ: 株式 / ファンド / ハイブリッド
   - リスクレベル表示（eToro風）
   - フォロワー数・リターン・更新日を一目で
2. **話題の銘柄**
   - $NVDA などタップで銘柄詳細へ（X Cashtag風）
3. **相場観**
   - 現状維持（強気/弱気バー）

### C. ポートフォリオカード強化（Trading212 + eToro）

| 項目 | 現状 | 改善 |
|------|------|------|
| パイチャート | 44px | そのまま or やや大きく |
| リターン | 1Y表示 | 1Y + リスクスコア（あれば） |
| コピー | 「参考にする」 | 「コピー」に統一し、ワンタップ感を強調 |
| 戦略タイプ | asset_type | 守備型 / 成長型 / バランス などラベル |

### D. 投稿 × 銘柄リンク（X Cashtag風）

- 本文中の `$7203` `$NVDA` をタップ可能に
- タップ → 銘柄詳細 or チャートへ遷移
- ハッシュタグと併用で「銘柄 × 議論」を強化

### E. ポートフォリオ変更通知（Trading212）

- 参考にしたポートフォリオの配分変更時に通知
- DB: `portfolio_follows` + `portfolio_allocations` の変更検知
- 通知: 「守備型グロ... の配分が更新されました」

### F. フィードタブの整理

| タブ | 役割 |
|------|------|
| For you | 人気投稿（現状維持） |
| Following | フォロー中の投稿 |
| Portfolio | **ポートフォリオ専用フィード**（Trading212 Social Pie風） |
| Q&A | 質問・議論 |
| Saved | 保存した投稿 |

→ Portfolioタブを「共有ポートフォリオの発見」に特化。

## 3. 実装優先度

| 優先度 | 項目 | 工数 | 効果 |
|--------|------|------|------|
| 高 | サイドバー: 戦略フィルタ・リスク表示 | 中 | 発見体験向上 |
| 高 | 投稿内 $銘柄 タップ → 銘柄詳細 | 中 | X風シームレス体験 |
| 中 | ポートフォリオ変更通知 | 大 | リテンション |
| 中 | PortfolioCard: 戦略ラベル・コピー文言 | 小 | 一貫性 |
| 低 | リスクスコア算出・表示 | 大 | 透明性 |

---

## 4. まとめ

**Trading212** の Social Pie（共有→コピー→変更通知）、**eToro** の戦略別発見と透明な指標、**X** の Cashtag とトレンドを組み合わせる構成がベスト。

MoneyMartの強みである「日本市場・ファンド」に合わせ、以下を軸にする：
1. **発見**: 戦略・リスクでフィルタ可能な人気ポートフォリオ
2. **議論**: 投稿 + $銘柄リンクで銘柄と議論を紐付け
3. **コピー**: ワンタップ「コピー」＋変更通知でフォロー体験を強化
